# VVV Custom site template
For when you just need a simple dev site

## Overview
This template will allow you to create a simple dev environment using only `vvv-custom.yml`.

The supported environments are:
- A single site

# Configuration

### The required configuration:

```yaml
my-site:
  repo: https://github.com/vahidm-sp/custom-site-template
  hosts:
    - my-site.test
```
